
# Commit Analyser

Commit Analyser to see which days of the week you commit the most and make a cool tagline with ChatGPT.

## How to use

1. Clone the repo
2. Run `pip install -r requirements.txt`
3. Run `python app.py`
4. Go to `localhost:5000` in your browser
